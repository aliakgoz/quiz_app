   versiyon kontrolü yapıyor.

4. REPOSITORY ====> the location where your code history is stored.
5. BRANCH ====> commitleri içeren klasör olarak düşünülebilir.
6. COMMITS ====> Yazdığımız Code un farklı versiyonları.

7. git init ====> boş bir repository oluşturur.
8. git status ====> repository durumunu gösterir. 
9. git add . ====> klasördeki tüm dosyaları takip sistemine alır.
10. git commit -m "commit name" ====> versiyon oluşturur.
11. git branch ====> branchları gösterir.
12.  git log ====> git in yaptığı işlemleri gösterir (q ile çıkılır)
